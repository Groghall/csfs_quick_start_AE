Module 2 - Mapping

What you'll need:

➢ Module 2 input file

➢ AEWS data validation spreadsheet

Summary

In this module we will be mapping payroll data items to AEWS so that regular input files are quickand easy to upload.

One of the key items in the mapping process is the group Group Code, this is unique to every group that is set up within AEWS and allows employees to be split across multiple groups with one file upload.

ABC Employers used the ledgers codes within their payroll system as the AEWS group codes below:

➢ Directors and Managers: 1234

➢ Warehouse Staff: 5678

Internal steps ABC Employer took

➢ Reviewed the AEWS Data Reguirements documetn with their payroll provider.

➢ Identified how they build the source data for AEWS Input files.

➢ Defined a process to exract the data items on a regular basis.

➢ Completed the Mapping Process.

Key Learning outcome

➢ Getting accurate and reliable data for AE has been a real challenge for many employers, you should start this process as soon as possible

➢ Group code cannot start with a leading zero.
