Introduction

In order to complete this guide you will require access to the AEWS Test Site. If you have not been provided with access then please contact the AE Support team by email - aesupport@csfs.co.uk or phone - 0151 255 2690. The team will provide you with a username and password information for logging into the middleware system located here.

The username and temporary password have been created by the AE support team and will be provided to you via email. The temporary password has a seven day lifecycle so please ensure that you complete module 1 within this timeframe. The test site has a timed access limit of 30 days from the day that you request access. This time limit can be extended upon request.

Please ensure that you use your time well and run through this guide completly to ensure a thorough understanding of the middleware system before commencing with a live site.

All the required documents for use within the test site are available within the downloadable zip folder here which we recommend that you download to a local folder/drive.

To complete this test of AEWS it should take approximatley two hours. It is recommended the guide is completed in it's entirety in one sitting where possible. We suggest you extract all files from the zip folder and save them to your local network.


The guide consists of nine modules as listed below:

    ➢ Group Set Up

    ➢ Mapping a payroll extract to AEWS

    ➢ Certification

    ➢ Compliance (5 steps)

    ➢ Settings (communications set up, attachments, branding)

Each module requires you to:

    ➢ Log into the AEWS Test Site (if not already logged in)

    ➢ Follow the necessary steps provided in the module.

    ➢ Provide evidence that you have completed the steps (activity files)

    ➢ Document your own internal company processes for future reference


Each module follows a case study of a small company named ABC Employer through 4 months of payroll activity from their staging date. The case study will provide you with all the necessay documentation to complete each module and will provide reference to the AEWS user guide when needed


Getting Started

It is presumed that you should already have a good understanding of AE and have defined a strategy for complying with your obligations.

Creating the following file structure on your local network will help you orgainse your comliance checks.

As we progress through the guide you will be asked to save the system outputs just as you would when using the real site. This forms part of your Auto Enrolment audit history.


Make sure that you have access to the AEWS test site and have completed your login / security details before continuing.

AEWS produces documentation that can be printed and posted or that can be emailed as an attachment. If you wish to view the email functionality please enter your email address in the AEWS input files as opposed to using your employees.
