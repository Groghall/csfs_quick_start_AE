Module 2 - Mapping

Steps for you to complete continued
  1. Map the column headers

    1. Click on a data item under the 'Data we need' header
    2. Click on the same item under the 'Your Files's Field' header
    3. Click on 'Link to' to map.

    Continue this process until all data items are mapped.



    2. Although ABC Employer has reviewd the Data requirements document there is 1 error with their upload file. The error report will only provide ABC Employer with a list of the employee lines with errors not the entire AEWs input file



  2. Download the error file



  3. Open the Module 2 file and make the necessary amendments and save

  4. Re-upload the amended Module 2 file

  5. Repeat steps 5 and 6

  6. Select 'Done' and 'Submit Company Setup'

We have now successfully mapped ABC employers AEWs input file and dealt with the errors within the file.
