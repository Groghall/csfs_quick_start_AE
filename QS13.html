Summary

In this module we will run month 3 compliance where we will see how AEWS deals with the eligible jobholders that have reached the end of their postponement period (issued month 1) and with employees that have previously been processed for an Opt In.


During this month several employees had a pay rise which resulted in them changing status, AEWS monitors both existing and non existing members for upward status changes and then displays the relevant compliance action needed.
