Module 1 - Group Set Up (Pension Data)

Steps for you to complete continued
  1. Select 'Start Pension Data' as indicated in the image below.



  2. Enter the staging date as stated within the fact find.



  3. Complete the scheme and group set up as per the fact find.

  4. Select 'Submit Details'

We have now successfully added the AE strategy to AEWS. This is the base information needed to run certification and compliance.

In further sections we will be uploading an AEWS input.


Key Learning outcome

➢ You need to make sure that the Pay Reference Period (PRP) in your group contains your staging date. E.g. The staging date 01/04/2015 is covered in their 10/03/2015 - 09/04/2015 pay reference period.

➢ The AEWS group codes are the link betwenn your payroll system and AEWS. These need to be a code you can control within the payroll system. This could be your payroll ledger codes (see section 4 of the user guide for more information on group codes).
