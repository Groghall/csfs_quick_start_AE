What you'll need:

    ➢ Module 2 file

    ➢ Internal Process Timeline

Summary

We will now begin our compliance modules. Throughout the next four modules we will demostrate the different scenarios you may get during your regualar compliance runs. We will also show how AEWs deals with different actions and the impact some actions have in future pay reference periods.

Within this module we will run through the staging month for ABC Employer.

Remember, during staging AEWS will be producing multiple types of doumentation depending on the worker status. AEWS operates on a tailored postponement basis and therefore only eligible jobholders will receive a postponement notice. AEWS will produce both existing scheme meber and non-scheme member communications to ensure you fulfill your regulatory obligations.

Internal Steps ABC Employer took

➢ Established a regular process for getting the data AEWS needs from their payroll

➢ Identified timelines for updates to payroll prior to payroll being run

➢ Endured that the existing member indicator within AEWS input file was correct

Key learning outcome

➢ In AEWS you can customise communications, more information can be found in the user guide section 8.2

➢ Using AEWS needs to fit into your payroll process to help you be compliant

Steps for you to complete
  1. Log into AEWS Test Site

  2. Select 'Start Compliance Check'

  3. Select 'Yes' to use the file uploaded in Module 2 and select the PRP that covers the staging date
